黑镜：潘达斯奈基 互动播放器

将视频文件拖动到此方框内；
或单击方框内任意位置，选择时长为5:12:40的《黑镜：潘达斯奈基》视频文件 视频文件选择完成。
  • 基于Chromium内核的浏览器（Google Chrome,Microsoft Edge【强烈推荐】,Opera,搜狗浏览器等）支持mkv和mp4；Firefox火狐浏览器仅支持mp4格式。
  • 经测试，谷歌Chrome 99.0.4844.51及FireFox 97.0.2支持x264视频编码及AAC音频编码，不支持x265视频编码及杜比AC3音频编码；
    微软Edge浏览器 99.0.1150.30 除了x264和AAC外，也支持x265及AC3；其他浏览器未测试，这取决于浏览器对html5音视频编码的支持以及你电脑上的解码器。
  • 离线播放时 ，由于浏览器的安全限制，Chrome等浏览器默认不可以通过file协议加载文件，从而导致 无法加载字幕文件 。
    1. 先关闭所有浏览器窗口后，直接双击打开文件夹内的自动播放.bat文件 ，会在几秒后(有一定的启动时间)自动新打开一个可以选择字幕的浏览器。
    2. Chrome浏览器，可以先关闭所有Chrome浏览器窗口，然后再在CMD内start chrome --allow-file-access-from-files --allow-file-access --allow-cross-origin-auth-prompt 参数启动浏览器，最终在此新启动的浏览器界面内打开对应的html文件。
    3. 微软Edge浏览器，可使用start msedge --allow-file-access-from-files --allow-file-access --allow-cross-origin-auth-prompt来启动，最终在此新启动的浏览器界面内打开对应的html文件。
    4. FireFox浏览器，则需要在地址栏输入about:config来打开高级设置，然后搜索privacy.file_unique_origin双击选项，将设置从True调整为False，然后刷新网页即可。（默认情况下，FireFox显示的字幕可能不太大，可以通过【Crtl+鼠标滚轮】调节字体大小，【Ctrl +0】 恢复默认字体大小。）
  • 请自行下载时长为5:12:40的《黑镜：潘达斯奈基》视频文件。 本播放器已经Black.Mirror.Bandersnatch.2018.1080p.NF.WEB-DL.x264.AAC-PTerWEB.mp4 测试通过。
  • 默认简英双语字幕，支持简英双语，简体中文，繁體中文，English SDH，Arabic，Danish，German，Greek，Spanish(es)，Spanish(es-ES)，Finnish，French，Hebrew，Indonesian，Icelandic，Italian，Japanese，Korean，Norwegian (Bokmål)，Dutch，Polish，Portuguese (Portugal)，Portuguese (Brazil)，Romanian，Russian，Swedish，Thai，Turkish，Vietnamese等多国字幕。
  • 切换字幕的方法：1.播放视频后，在视频内鼠标右键打开菜单. 2.启用【显示所有控件. 3.鼠标单击页面右下角的 4.选择对应的字幕. 5.然后按照上边步骤1-2关闭【显示所有控件】
  • 播放器的控制面板可以通过上边的【显示所有控件】打开，可以选择字幕、播放速度等。请勿在全屏播放时持续启用【显示所有控件】功能，使用完成后立即关闭！！！ 否则可能遇到播放问题。

  • F / F11 / 鼠标双击 - 开关全屏模式
  • 空格 / 鼠标单击 - 切换播放/暂停
  • → / ← - 跳到下一个/上一个 剧情支线(或互动选择界面)
  • ↑ / ↓ - 加速播放/减速播放
