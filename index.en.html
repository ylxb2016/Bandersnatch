  • Chromium based Browser(Google Chrome,Microsoft Edge,Opera, etc.) support MP4 and MKV, while Firefox only MP4.
  • Microsoft Edge(99.0.1150.30) support x265/x264 codec and AAC/AC3 codec, while Chrome(99.0.4844.51) or Firefox(97.0.2) only x264 and AAC codec.
  • When Playing offline,due to browser security restrictions, browsers cannot load files through the file:// protocol directly, resulting in failing to load subtitles files .Here are some solution:
    1. Close all the browser tabs, then open the autoplay.bat
    2. Chrome: Close all the Chrome tabs，and start the Chrome by CMD with start chrome --allow-file-access-from-files --allow-file-access --allow-cross-origin-auth-prompt, then open the index.html with it.
    3. Microsoft Edge: same as Chrome, but with start msedge --allow-file-access-from-files --allow-file-access --allow-cross-origin-auth-prompt
    4. FireFox: enter about:config in the address bar to open Advanced Preferences,accept the risks,then searchprivacy.file_unique_origin in the list using the search bar,double click on the line to pass the value to False.
